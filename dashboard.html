DARWIN (Dynamic Adaptive Risk-Weighted Intelligence Network) — 一个由 Claw 驱动、基于 OKX Agent Trade Kit 和 Paperclip AI 构建的本地优先、风险优先的 AI 交易治理系统。

Local-first, risk-first AI trading governance system powered by Claw, OKX Agent Trade Kit, and Paperclip AI.

它把市场识别、策略治理、真实执行、风控熔断和审计报告保持在同一条决策轨道上。

技术栈: Claw / OKX Agent Trade Kit / Paperclip AI / Node.js 22 / TypeScript / SQLite / Optional LLM providers
